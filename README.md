## 一些基础中的基础
### Compiler
计算机很笨，只能理解一种语言就是机器语言(由0和1组成). 为了让人和机器都相互理解对方，人类开发出了高级语言(high level programming language 比如 java, C++). 

把高级语言 翻译成 机器语言 的程序就是编译器(compiler).
